# Expense Tracker App

## Description
This is a simple Expense Tracker application built using React.  
It allows users to track their expenses, view total spending, and analyze expenses by category.

## Features
- User registration and login
- Add new expenses with amount, category, and date
- View total and monthly expenses
- Filter expenses by category and date
- View expense statistics with charts
- Data stored using browser localStorage

## Tech Stack
- React
- Redux Toolkit
- React Router
- JavaScript
- Bootstrap

## How to Run the Project
1. Clone the repository  
2. Run `npm install`  
3. Run `npm run dev`

## Project Type
Frontend React Project (No backend)
